# دانلودر یوتیوب Flutter Stuff

ساخته شده توسط Farzin

[English README](README.md)

یک دانلودر مدرن برای ترمینال که با Python، `yt-dlp` و Rich ساخته شده است. این برنامه می‌تواند ویدیو یا پلی‌لیست یوتیوب را دانلود کند، کیفیت ویدیو یا حالت صوتی را انتخاب کند، پیشرفت دانلود را زنده نشان دهد و در پایان مسیر ذخیره فایل‌ها را نمایش دهد.

## امکانات

- رابط ترمینالی مدرن با لوگوی بزرگ `Flutter Stuff`
- پشتیبانی از دانلود ویدیو تکی و پلی‌لیست
- انتخاب کیفیت ویدیو
- دانلود صوتی به صورت MP3، M4A یا فرمت اصلی
- دانلود همزمان چند فایل در پلی‌لیست
- نمایش زنده وضعیت، سرعت، حجم، درصد پیشرفت و زمان باقی‌مانده
- امکان توقف موقت، ادامه و لغو دانلود
- نمایش مسیر ذخیره فایل‌ها در پایان دانلود

## پیش‌نیازها

- Python نسخه 3.10 یا جدیدتر
- نصب بودن `ffmpeg` و در دسترس بودن آن در `PATH`
- اتصال اینترنت

برای ترکیب صدا و تصویر و تبدیل صوت به MP3 یا M4A به `ffmpeg` نیاز دارید.

فایل‌های `install.bat` و `install.sh` وابستگی‌های پایتون را نصب یا به‌روزرسانی می‌کنند و وضعیت FFmpeg را بررسی می‌کنند. اگر FFmpeg وجود نداشته باشد، برنامه تلاش می‌کند سورس FFmpeg را از GitHub با نمایش پیشرفت دانلود دریافت کند. اگر Git نصب نباشد، سورس به صورت فایل ZIP دانلود می‌شود. مخزن رسمی FFmpeg در GitHub شامل سورس کد است، بنابراین اگر بعد از دانلود فایل اجرایی `ffmpeg` پیدا نشود، نصب‌کننده به کاربر هشدار می‌دهد.

فایل‌های `run.bat` و `run.sh` چیزی نصب نمی‌کنند. آن‌ها فقط در صورت نبودن FFmpeg یا وابستگی‌های پایتون به کاربر هشدار می‌دهند.

## نصب

ویندوز:

```powershell
.\install.bat
```

لینوکس یا macOS:

```sh
chmod +x install.sh run.sh
./install.sh
```

## اجرا

ویندوز:

```powershell
.\run.bat
```

لینوکس یا macOS:

```sh
./run.sh
```

اجرای مستقیم با پایتون:

```sh
python yt_downloader.py
```

اجرای مستقیم با لینک:

```sh
python yt_downloader.py "VIDEO_URL" --concurrency 2
```

دانلود فقط صوت:

```sh
python yt_downloader.py "VIDEO_URL" --audio
```

دانلود فقط یک ویدیو از لینک پلی‌لیست:

```sh
python yt_downloader.py "PLAYLIST_OR_VIDEO_URL" --no-playlist
```

## محل ذخیره فایل‌ها

به صورت پیش‌فرض فایل‌ها در این مسیرها ذخیره می‌شوند:

```text
Downloads/yt_downloader/video
Downloads/yt_downloader/audio
```

برای انتخاب مسیر دیگر:

```sh
python yt_downloader.py "VIDEO_URL" --output ~/Videos
```

در پایان دانلود، برنامه مسیر دقیق ذخیره فایل‌ها را نشان می‌دهد. اگر ترمینال شما از لینک پشتیبانی کند، با کلیک روی مسیر می‌توانید پوشه را باز کنید.

## کنترل‌ها هنگام دانلود

- `p` توقف موقت یا ادامه دانلود
- `c` لغو دانلود
- `q` لغو دانلود

## نکات

- دانلود پلی‌لیست می‌تواند چند فایل را همزمان انجام دهد.
- اگر دانلود را متوقف کنید، همان فرایند بعد از ادامه دادن از همان‌جا ادامه پیدا می‌کند.
- اگر دانلود لغو یا قطع شود، اجرای دوباره همان دستور معمولا فایل نیمه‌کاره را ادامه می‌دهد؛ البته تا وقتی یوتیوب و فرمت انتخاب‌شده اجازه بدهند.
- بعضی ترمینال‌ها از لینک قابل کلیک پشتیبانی نمی‌کنند. در این حالت مسیر کامل پوشه همچنان نمایش داده می‌شود و می‌توانید آن را دستی باز کنید.
